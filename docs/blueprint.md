# **App Name**: Thapong Arts Hub

## Core Features:

- Public Marketplace & Browsing: Visitors can browse artworks (filtered by price, artist, category, availability), artist profiles, events, and creative businesses without login. Each entry features an image, title, and key details.
- Artist Dashboard for Content Management: Authenticated artists can add, edit, or delete their artworks, upload images to Supabase Storage, set pricing and availability status. They can also manage their public profile details and create events.
- Secure Authentication and Authorization: Leverages Supabase Auth for artist and admin roles, ensuring secure access to dashboards and enforcing Row Level Security (RLS) policies on the Supabase database for data protection and management.
- Direct Artist Engagement via WhatsApp: Artwork and artist profile pages feature a dedicated WhatsApp button to enable direct communication between interested buyers and artists, pre-populating a message with artwork details.
- AI-Powered Artwork Description Tool: An integrated AI tool assists artists in crafting compelling and unique descriptions for their artworks, taking basic input and generating engaging narratives.
- Dynamic SEO & Metadata Generation: Automatically generates SEO-friendly titles and meta descriptions for all artwork and artist profile pages, optimizing discoverability for search engines.
- Supabase Backend Integration: Comprehensive integration with Supabase for data management (artists, artworks, events, businesses), file storage (artwork images via Supabase Storage), and robust Row Level Security.

## Style Guidelines:

- Primary Color: A sophisticated deep indigo (#41367D), offering an elegant contrast on light backgrounds and reflecting creativity.
- Background Color: A subtle, near-white #F4F3F7, with a hint of the primary hue, providing a clean, minimalist canvas that allows artwork to be the central focus.
- Accent Color: A vibrant medium blue (#3C83ED) for interactive elements and call-to-actions, providing clear visual guidance while complementing the primary tone.
- Headline Font: 'Space Grotesk' (sans-serif), for a modern, slightly tech-inspired, yet clean aesthetic. Body Font: 'Inter' (sans-serif), chosen for its excellent readability and neutral, objective appearance in longer text blocks.
- Utilize minimalist, elegant line-art or geometric icons to maintain a sophisticated visual harmony and not detract from the artwork presented.
- Implement a 'modern art gallery' layout characterized by generous whitespace, large, impactful imagery for artworks, and a grid-based system for easy browsing and visual hierarchy.
- Incorporate subtle, smooth transitions and micro-interactions on hover states or navigation to enhance the user experience without being distracting, mirroring the app's elegant design.
